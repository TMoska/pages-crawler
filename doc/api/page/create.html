Page API

CREATE

POST /api/pages

Reads contents of an url and stores information from h1, h2, h3 and a HTML tags.

Parameters

Name Description
url URL of a website

Response Fields

Name Description
id ID of resource
url URL of a passed website
headers Header resources
headers[:text] Content of a heading
headers[:level] Identifies if its h1 h2 or h3
links Link resources
links[:text] Content of a link

Headers

Host: example.org
Content-Type: application/x-www-form-urlencoded
Cookie:

POST /api/pages

url=http%3A%2F%2Frandomusefulwebsites.com
